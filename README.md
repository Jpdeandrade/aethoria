# Aethoria: Living Universe

> **"Where Every Game Writes History"**

Um universo de RPG vivo, construído por mesas reais. Jogue qualquer sistema. Seus eventos viram lore oficial.

🌐 **[aethoria.online](https://aethoria.online)**

---

## O que é

Aethoria é uma plataforma de universo compartilhado para jogadores de RPG de qualquer sistema. Campanhas independentes de D&D, Call of Cthulhu, Pathfinder e outros sistemas coexistem no mesmo mundo — e as ações de uma mesa afetam a lore que todos compartilham.

**Não é** uma ficha digital, VTT ou calculadora de mecânicas.
**É** uma plataforma de narrativa persistente, cross-system, curada por staff.

---

## Por que existe

A comunidade de RPG é fragmentada. Cada mesa é uma ilha. Personagens morrem com as campanhas. Mundos criados por mestres existem só para 4–6 pessoas. Sistemas diferentes simplesmente não conversam.

Aethoria resolve isso com três pilares:

- **Universo Vivo** — eventos de mesas reais mudam Aethoria permanentemente
- **Multi-Sistema** — D&D, CoC, Pathfinder e outros no mesmo universo
- **Curadoria Ativa** — staff valida eventos antes de virarem cânone

---

## Como funciona

1. Jogador cria um personagem vinculado a um sistema (imutável)
2. Mestre abre uma missão no quadro — filtrada por sistema
3. A sessão acontece fora da plataforma (Discord, presencial, Roll20...)
4. Mestre registra o resultado: níveis, itens, proposta de evento à curadoria
5. Staff valida e publica — o evento entra na lore oficial para sempre
